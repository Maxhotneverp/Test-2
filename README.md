<h2>通缩燃烧规则说明</h2>
		
                       每交易一笔将燃烧10%代币
                       例：某某购买1000枚代币，实际到账900枚代币（其中5%代币=50枚永久销毁，另外5%代币=50枚进入奖金池）
                       ⚠️：其中买入扣除的10%买入者买入时直接扣除，卖出扣除的10%不为交易者承担计入大盘总流通筹码的定向数据通缩 
 

<h2>代币供给体系说明</h2>
                       SWAP底仓：30000000  
                       弹性供给仓：无上限！   
<h4>弹性供给仓：</h4>
                       （Z为持币基准持仓数值，P为价格，T为结算时间,S为弹性供给数量）
                       S = Z * {（P2-P1）/ P1} / 10 /30
                       ⚠️：{（P2-P1）/ P1} 部分只取值《=10%部分，溢出部分不计算！
